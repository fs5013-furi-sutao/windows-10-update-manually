# アップデートツールで Windows 10 Home を手動更新する方法


## ダウンロードページを開く

Windows 10 のダウンロード  
https://www.microsoft.com/ja-jp/software-download/windows10  

「今すぐアップデート」をクリック

![](./windows_10_download.png)

## アップデートツールの実行

保存したダウンロードファイル `Windows10Upgrade{id}.exe` を実行

セキュリティの警告が表示されたら「実行」をクリック

「Windows 10 の最新バージョンへの更新」が表示されたら「今すぐ更新」をクリック

「このPCはWindows10と互換性がありあます」と表示されたら、「次へ」をクリック

`自動的にダウンロード開始`
